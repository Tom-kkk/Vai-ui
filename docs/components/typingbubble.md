# TypingBubble 打字气泡

用于展示 AI 正在输入或思考状态的动画气泡。

<demo vue="../demos/typingbubble.vue" />

## API

### Attributes

| 属性名 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| text | 提示文本 | `string` | `'AI 正在输入...'` |
| size | 气泡大小 | `'small' \| 'medium' \| 'large'` | `'medium'` |
| color | 主题颜色 | `string` | `'#1890ff'` |
| speed | 动画速度 | `number` | `1.4` |
| dots | 动画点数量 | `number` | `3` |
